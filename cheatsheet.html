8. Command cheatsheet¶

This section provides a quick summary of the data acquisition commands discussed in the previous sections.

8.1. Command summary¶

Table 8.1 Main BlueSky commands used at BMM (don’t type the < or >, those symbols indicate that the argument is numeric.)¶
          Shutter commands
shb.open() / shb.close() Open / close the photon shutter
          Mono tuning commands
tu() / td() Tune the mono 2nd crystal
TUNE_STEP=0.004 Tuning step size – 0.004 is good for Si(111), 0.002 for Si(311)
          Mono movement commands
RE(mv(dcm.energy, <value>)) Move TO an energy value
RE(mvr(dcm.energy, <value>)) Move BY an energy step
          Motor movement commands
RE(mv(xafs_liny, <value>)) Move motor TO a position
RE(mvr(xafs_liny, <value>)) Move motor BY an amount
RE(slot(<N>)) Move sample wheel to slot #N
          Where are things?
%w dcm where’s the mono?
%w slits3 where are the slits?
%w m2 where’s mirror 2? (toroidal focusing mirror)
%w m3 where’s mirror 3? (flat, harmonic rejection mirror)
%w xafs_table where’s the XAFS table?
          Change edge, mode, and mono
RE(change_edge('Xx')) setup beamline for an absorption edge, 1- or 2-letter symbol
RE(change_mode('X')) set mirror mode, X ∈ (A,B,C,D,E,F), see table below
RE(change_xtals('h11')) set monochromator, Si(111) or Si(311), h=1 or h=3
          Mono and experiment setup
RE(rocking_curve()) optimize pitch of 2nd DCM crystal (moves automagically)
RE(rocking_curve(detector='Bicron')) same, but for XRD mode
RE(slit_height()) explore position of slits3 (then pluck to move dm3_bct)
          Get help
%m Show motors
%h Show help
%k Show keyboard shortcuts
%se Show reference foil and ROI configurations

8.2. Photon delivery system modes¶

Table 8.2 Photon delivery system modes¶
Mode focused energy range
A ✓ 8 keV and up
B ✓ below 6 keV
C ✓ 6 keV to 8 keV
D ✗ 8 keV and up
E ✗ 6 keV to 8 keV
F ✗ below 6 keV
XRD ✓ 8 keV and up

8.3. Slits3 coordinated motions¶

These coordinated motions behave just like single motors and can be used with the motor movement commands in Table 8.1.

Horizontal size
slits3.hsize (nominally 8 mm)
Horizontal center
slits3.hcenter (nominally 0 mm)
Vertical size
slits3.vsize (nominally 1 mm)
Vertical center

slits3.vcenter (nominally 0 mm)

The vertical center is normally not changed. Instead, align the slits using dm3_bct and the slit_height() plan

8.4. Motor positions and limits¶

Where is a sample motor?
%w xafs_x
What are the soft limits?
xafs_x.hlm.value / xafs_x.llm.value
Set a soft limit:
xafs_x.hlm.put(-95) / xafs_x.llm.put(-157)

8.5. Line scans¶

RE(linescan(<detector>, <motor>, <start>, <stop>, <N>))

where

  • <detector> is one of: 'it', 'if', 'ir', or 'i0'
  • <motor> is one of: 'x', 'y', 'pitch', 'wheel', or a motor name
  • <start>, <stop>, <N> are the boundaries relative to the current position and the number of steps.

The plot will be determined from the values of <motor> and <detector>

This is a relative scan.

After prompt, single click the left button after a linescan to move to a position.

RE(pluck())

to repeat that on the current plot. RE(pluck()) only works on the most recent plot.

8.6. Energy scans¶

RE(xafs('myscan.ini'))

In the INI file, set mode to transmission, fluorescence, reference, or both to control the in-scan plotting display (both = show transmission and fluorescence)

8.7. Experiment log¶

Log entries are made for each scan. System and beamtime specific logs are maintained. To insert a comment in the log, do:

BMM_log_info(“This is my log entry”)
