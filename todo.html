10. To Do List¶

This is an aggregate list of things missing from the BMM beamline documentation manual.

  1. Document change_edge()

  2. Document pseudo channel cut & fixed exit DCM modes

  3. Document use of XDI_record to control what motors get recorded in the XDI header

This is an aggregate list of things mentioned in this document that the BMM data collection system needs to do.

  1. Better sanity checking on linescan input

  2. Better sanity checking on areascan input

  3. Have database consume snapshots

  4. Mode C lookup table

  5. Lookup table for E < 8 keV at goniometer

  6. Look up E0 from element & edge, use E0 to specify a non-edge energy

  7. Better sanitizing of INI file input

  8. Better heuristics for scan time remaining

10.1. Add a scan.ini parameter¶

  1. Add its entry into scan_metadata() in 84-energyscan.py

  2. Add its default value to the BMM_configuration class in 74-modes.py

  3. Add it to html_dict in 84-energyscan.py around line 912

  4. Correspondingly, add to the call arguments of scan_sequence_static_html() in 84-energyscan.py

and, of course, use the new parameter in whatever way it needs to be used.
